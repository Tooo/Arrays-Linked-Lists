# Arrays and Linked Lists in C++

## Installation
1. Clone the repository
```bash
git clone git@github.com:Toooo123/Arrays-Linked-Lists.git
```
2. Make the objects and executable file: 
```bash
make
```
3. Run the executable:
```bash
main
```
